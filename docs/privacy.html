What we collect

Custard Calendar is a free tool for checking frozen custard schedules. Here is what the site does and does not collect.

No accounts, no cookies

There is no login, no registration, and no persistent cookies. Each visit is independent. Nothing is stored in your browser except the store you selected (saved to localStorage on your device only, never sent to a server).

Anonymous usage telemetry

To understand which features are useful, the site sends anonymous interaction signals when you click certain buttons (Directions, Set Alert, Subscribe to Calendar) or complete a quiz. Each signal includes:

  • A random session ID generated fresh each page load (not tied to you across sessions)
  • The action you took (e.g., "directions click")
  • Your approximate country/region (from Cloudflare's edge network, not GPS)
  • A timestamp

No IP address is stored. No name, email, or device identifier is collected. The session ID resets on every page load — there is no persistent user ID.

Alert email subscriptions

If you subscribe to flavor alerts on the Alerts page, your email address is stored to send you notifications. You can unsubscribe at any time using the link in any alert email. We do not share email addresses with third parties.

Flavor data

Flavor schedules are fetched from the public websites of participating restaurants (Culver's, Kopp's, Gille's, Hefner's, Kraverz, Oscar's). Custard Calendar is not affiliated with any of these restaurants.

Third-party services

  • Cloudflare Workers — hosts the API and processes requests. Subject to Cloudflare's privacy policy.
  • GitHub Pages — serves these HTML pages. Subject to GitHub's privacy policy.
  • Google Calendar / Apple Calendar — optional integrations you initiate by clicking Subscribe. These services have their own privacy policies.
