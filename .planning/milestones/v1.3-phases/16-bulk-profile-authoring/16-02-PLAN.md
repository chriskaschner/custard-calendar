- cd custard-calendar/worker && npx vitest run palette-sync.test.js -x

Density guide:
- pure: no mix-ins (e.g., plain Vanilla)
- standard: 1-3 mix-in types
- double: "Double" in name or heavy single-ingredient
- explosion: 3+ distinct mix-in types
- overload: "Overload" in name, massive single-topping
