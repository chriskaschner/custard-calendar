**engine.js changes (in the multiple_choice block, around lines 474-512):**

After the grid element is created and before options are appended, detect whether ALL options in the current question have icon fields. If so, add 'quiz-image-grid' class to the grid element. This must check ALL options first, not add/remove per-option:

```javascript
// After: grid.className = 'quiz-options-grid';
// Add: check if ALL options have icons
var allHaveIcons = question.options.every(function(opt) {
  return opt.icon && window.QuizSprites;
});
if (allHaveIcons) {
  grid.classList.add('quiz-image-grid');
}
```

For icon rendering in image-grid mode, use a larger scale (6 instead of 4) so icons are visually prominent in the grid cell:

```javascript
var iconScale = allHaveIcons ? 6 : 4;
var iconSvg = option.icon && window.QuizSprites
  ? window.QuizSprites.resolve(option.icon, iconScale) : '';
```

No other changes to engine.js. The existing icon rendering logic (has-icon class, quiz-option-icon span, quiz-option-label span) remains as-is.
