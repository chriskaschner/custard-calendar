**Deploy accepted PNGs:**

Read the final ai-generation-manifest.json (either from the Export Manifest download or from the file updated by the gallery). For each flavor with status "approved" and a non-null "selected" field:

1. Copy the selected processed candidate from `docs/assets/ai-candidates/{slug}/{selected}` to `docs/assets/cones/{slug}.png` (overwriting the existing SVG-rasterized PNG)
2. The deployed file should be the `-processed.png` version (288x336, optimized)

Do this with a simple Node.js script or shell loop:

```bash
node -e "
const fs = require('fs');
const path = require('path');
const manifest = JSON.parse(fs.readFileSync('docs/assets/ai-generation-manifest.json', 'utf8'));
let copied = 0;
for (const [key, entry] of Object.entries(manifest)) {
  if (entry.status === 'approved' && entry.selected) {
    const src = path.join('docs/assets/ai-candidates', entry.slug, entry.selected);
    const dst = path.join('docs/assets/cones', entry.slug + '.png');
    fs.copyFileSync(src, dst);
    copied++;
  }
}
console.log('Deployed ' + copied + '/94 PNGs to docs/assets/cones/');
if (copied !== 94) { console.error('WARNING: Expected 94, got ' + copied); process.exit(1); }
"
```
