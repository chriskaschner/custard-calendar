**A) Create `tools/generate_cone_art.mjs`** -- Azure OpenAI image generation script for cone art.

API key retrieval (in this order):
1. `process.env.AZURE_OPENAI_API_KEY` if set
2. Otherwise: `execSync('security find-generic-password -a "$USER" -s "azure-openai-api-key" -w').toString().trim()` (import execSync from 'node:child_process')
3. If neither available, print error and exit 1

Constants:
```javascript
const ENDPOINT = 'https://etc-ai-foundry-sbx-east-us-2.cognitiveservices.azure.com/openai/deployments/gpt-image-1.5/images/generations?api-version=2024-02-01';
const CANDIDATES_DIR = path.join(REPO_ROOT, 'docs', 'assets', 'ai-candidates');
const FLAVOR_FILLS_PATH = path.join(REPO_ROOT, 'docs', 'assets', 'masterlock-flavor-fills.json');
const MANIFEST_PATH = path.join(REPO_ROOT, 'docs', 'assets', 'ai-generation-manifest.json');
const DELAY_MS = 5000; // 5 seconds between requests (conservative default)
```

Prompt assembly function `buildL5Prompt(flavor)`:
- Start with the MASTERLOCK_TEMPLATE from the flavor fills JSON
- Replace `Quality target:\nChoose one tier from the complexity ladder: [L0 | L1 | L2 | L3 | L4 | L5]\nDo not blend tiers in a single render.` with `Quality target: L5 -- Premium Showcase`
- Replace the `Background:` section (from "Background:" through "No text.") with:
  ```
  Background:
  Transparent background.
  No gradients. No texture. No logo. No text.
  Ensure there is no transparency within the ice cream scoop or waffle cone itself, only around the isolated subject.
  ```
- Replace the `Flavor:` section and everything below it with the flavor-specific fill using `premium_treatment_override` (NOT `ingredient_treatment`):
  ```
  Flavor: {flavor.title}

  Description:
  {flavor.description}

  Ingredient treatment:
  - Base custard color: {flavor.premium_treatment_override.base}
  - Swirls: {flavor.premium_treatment_override.swirls}
  - Chunk inclusions: {flavor.premium_treatment_override.chunks}
  - Texture notes: {flavor.premium_treatment_override.texture}
  ```

Generation function `generateImage(apiKey, prompt, quality)`:
- POST to ENDPOINT with headers `{ 'api-key': apiKey, 'Content-Type': 'application/json' }`
- Body: `{ prompt, n: 1, size: '1024x1024', quality, background: 'transparent', output_format: 'png' }`
- Handle 429 rate limit: read `retry-after` header, wait that many seconds, retry once
- On success: return `Buffer.from(json.data[0].b64_json, 'base64')`
- On other errors: throw with status code and response text

Main function with CLI flags:
- `--trial` -- Generate 3 candidates each at BOTH medium and high quality for 3 trial flavors (18 images total). Trial flavors: `vanilla`, `mint explosion`, `caramel turtle`.
- `--quality medium|high` -- Quality setting for full batch generation (required for non-trial mode)
- `--flavor "flavor key"` -- Generate 3 candidates for a single specific flavor (for targeted regeneration)
- `--all` -- Generate 3 candidates for all 94 flavors at the specified --quality

For each flavor generated:
1. Create directory `docs/assets/ai-candidates/{slug}/`
2. Generate 3 images, saving as `{slug}-1.png`, `{slug}-2.png`, `{slug}-3.png` (for trial: also `{slug}-medium-1.png` etc. and `{slug}-high-1.png` etc.)
3. Wait DELAY_MS between API calls
4. Write/update `docs/assets/ai-generation-manifest.json` with entry per flavor: `{ flavor_key, slug, model: 'gpt-image-1.5', quality, size: '1024x1024', background: 'transparent', prompt_hash: sha256(prompt), candidates: [{file, generated_at}], selected: null, status: 'pending' }`
5. Print progress: `[{n}/{total}] {slug}-{candidate}.png ({quality})`

The manifest is an object keyed by flavor_key. Load existing manifest if present, merge new entries.

For `prompt_hash`: use `crypto.createHash('sha256').update(prompt).digest('hex').slice(0, 16)`.

**B) Create `scripts/postprocess_ai_cones.mjs`** -- Sharp-based post-processing pipeline.

Import sharp using the same fallback pattern as generate-hero-cones.mjs (try import('sharp'), fallback to worker/node_modules require).

CLI modes:
- `node scripts/postprocess_ai_cones.mjs` -- Process all files in docs/assets/ai-candidates/*/
- `node scripts/postprocess_ai_cones.mjs --slug {slug}` -- Process only files for one flavor
- `node scripts/postprocess_ai_cones.mjs --output-dir {path}` -- Override output directory (default: writes processed files in-place alongside originals with `-processed` suffix)

Processing pipeline per image:
```javascript
await sharp(inputPath)
  .trim({ threshold: 5 })          // Conservative trim
  .resize({
    width: 288,                      // 2x retina (144x2)
    height: 336,                     // 2x retina (168x2)
    fit: 'contain',
    kernel: 'nearest',               // Preserve pixel art crispness
    background: { r: 0, g: 0, b: 0, alpha: 0 },
  })
  .png({
    compressionLevel: 9,
    palette: false,                  // Keep full RGBA
  })
  .withMetadata({ density: 300 })
  .toFile(outputPath);
```

Output naming: `{slug}-1-processed.png` alongside the raw `{slug}-1.png`.

Print per-file: `[{n}/{total}] {filename} -> {output} ({originalSize} -> {processedSize} bytes)`
